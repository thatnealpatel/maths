1 — Kernel Functions

A kernel function K(x, y) measures similarity between two points. The RBF (radial basis function) kernel K(x, y) = exp(−||x−y||² / 2σ²) returns 1 when x = y and decays toward 0 as points move apart. Each sample xi drawn from distribution P creates a function K(xi, ·) — a Gaussian bump centered at that sample. These functions live in the RKHS, an infinite-dimensional function space where inner products correspond to kernel evaluations. The bandwidth σ controls how localized each bump is: small σ produces narrow spikes that resolve fine structure, while large σ creates broad humps that blur nearby samples together.

Try: set σ to 0.1 and watch the kernels become needle-thin. Then increase to 2.0 — individual samples merge into a single lobe. With n = 3 or 4 you can see each kernel clearly; with n = 50 they overlap into a dense forest.

2 — Mean Embedding

The mean embedding μP maps an entire probability distribution into the RKHS by averaging the kernel functions of its samples: μ̂P(t) = (1/n) Σi K(xi, t). This is the central idea — we represent a distribution not by its density, but by a single function in the RKHS. With a characteristic kernel like the RBF, this mapping is injective: distinct distributions produce distinct embeddings. As n increases, the empirical mean embedding converges to the true (population) mean embedding, which in turn approximates a smoothed version of the underlying PDF. The plot shows individual kernel functions faded in the background, their average in bold, and the true PDF for reference.

Try: compare n = 5 vs n = 100. With few samples the embedding is jagged and sample-dependent; with many it hugs the true PDF shape. Switch P to "bimodal" to see how the embedding faithfully captures both modes.

3 — MMD Comparison

Maximum Mean Discrepancy (MMD) is the RKHS distance between two mean embeddings: MMD(P, Q) = ||μP − μQ||H. It is computed entirely from kernel evaluations without ever estimating densities. When MMD ≈ 0 the two distributions are indistinguishable under this kernel; large MMD means the distributions differ. The plot shows μ̂P and μ̂Q side by side, with the pointwise absolute difference |μ̂P(t) − μ̂Q(t)| highlighting where they diverge. The title reports both MMD² (the quantity actually computed) and MMD (its square root, the metric).

Try: set P = Q = gaussian. MMD should be near zero — both embeddings overlap. Now change Q to "skewed" or "uniform" and watch the gap open up. Increase n to reduce sampling noise and get a cleaner separation.

4 — Gram Matrix

The Gram matrix Gij = K(xi, xj) is the n×n matrix of all pairwise kernel similarities among the samples from P. The diagonal is always 1 (a point is maximally similar to itself). Off-diagonal values reveal structure: if samples cluster into groups, the matrix shows bright blocks along the diagonal. For a unimodal distribution like a Gaussian the matrix tends to be smoothly graded; for a bimodal distribution you should see two distinct bright blocks. The Gram matrix is the object that kernel methods (SVM, kernel PCA, etc.) actually operate on — it encodes all the geometric information the kernel can see.

Try: set P = bimodal with n = 30–40 and look for the 2×2 block structure. Then switch to uniform — the matrix becomes more homogeneous. Decrease σ to sharpen the blocks (each sample only "sees" its nearest neighbors); increase σ to wash out all structure.

5 — Sigma Sweep

The bandwidth σ is the single most important hyperparameter. Too small, and the kernel functions are so narrow that every finite sample looks unique — MMD is dominated by noise. Too large, and everything is smoothed into a single blob — even genuinely different distributions become indistinguishable. This plot sweeps σ from 0.1 to 3.0 and shows MMD² and MMD at each value. The peak of the curve is the bandwidth at which the kernel is most sensitive to the actual distributional difference. In practice, methods like the median heuristic (σ = median of pairwise distances) aim to land near this sweet spot.

Try: compare gaussian vs bimodal — the peak is clear and wide. Then compare gaussian vs gaussian (same distribution): the curve should stay near zero everywhere, confirming there is no real difference to detect. Compare uniform vs skewed for an interesting asymmetric shape.
